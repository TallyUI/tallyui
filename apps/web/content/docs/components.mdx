## Overview

Tally UI components are React Native primitives that work on iOS, Android, and Web via Expo. They use the [trait layer](/docs/architecture) to read data from any connector and [Uniwind](https://uniwind.dev) for className-based styling.

Every component accepts a `className` prop for style overrides. Defaults are merged using `cn()` from `@tallyui/theme`, so your classes cleanly override the built-in styles. See [Design System](/docs/design-system) for details.

## Product Components

### ProductTitle

Displays the product name. Uses `getName()` from the active connector's traits.

```tsx
import { ProductTitle } from '@tallyui/components';

<ProductTitle doc={productDocument} className="text-lg font-bold" />
```

The same component reads `doc.name` for WooCommerce and `doc.title` for Medusa — the trait layer handles the mapping.

**Default classes:** `text-base font-semibold text-foreground`

### ProductPrice

Displays the product price with optional currency symbol. Handles sale prices automatically — regular prices use `text-price`, sale prices use `text-sale`.

```tsx
import { ProductPrice } from '@tallyui/components';

<ProductPrice doc={productDocument} currencySymbol="$" />
```

- WooCommerce: reads `doc.price` (string like `"599.99"`)
- Medusa: derives from `doc.variants[0].prices[0].amount` (integer in cents)

**Default classes:** `text-sm font-medium text-price` (or `text-sale` when on sale)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `doc` | `any` | required | RxDB product document |
| `currencySymbol` | `string` | `'$'` | Currency symbol to prepend |
| `className` | `string` | — | Override default styles |

### ProductImage

Displays the product's primary image. Returns `null` if no image is available.

```tsx
import { ProductImage } from '@tallyui/components';

<ProductImage doc={productDocument} size={80} className="rounded-lg" />
```

- WooCommerce: reads `doc.images[0].src`
- Medusa: reads `doc.thumbnail` or `doc.images[0].url`

**Default classes:** `rounded`

The `size` prop controls width and height via the `style` prop (dynamic values can't be Tailwind classes). The `className` prop handles everything else.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `doc` | `any` | required | RxDB product document |
| `size` | `number` | `60` | Image width and height in pixels |
| `className` | `string` | — | Override default styles |

### ProductSku

Displays a product's SKU with a configurable fallback.

```tsx
import { ProductSku } from '@tallyui/components';

<ProductSku doc={productDocument} />
<ProductSku doc={productDocument} fallback="N/A" />
```

**Default classes:** `text-xs text-muted`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `doc` | `any` | required | RxDB product document |
| `fallback` | `string` | `'—'` | Text when SKU is missing |
| `className` | `string` | — | Override default styles |

### ProductStockBadge

Visual stock status indicator with colored dot and label. Uses semantic status colors from the design system.

```tsx
import { ProductStockBadge } from '@tallyui/components';

<ProductStockBadge doc={productDocument} />
<ProductStockBadge doc={productDocument} showQuantity />
```

Renders a pill-shaped badge with status-appropriate colors:
- **In Stock** — `bg-success/15 text-success`
- **Out of Stock** — `bg-danger/15 text-danger`
- **On Backorder** — `bg-warning/15 text-warning`
- **Unknown** — `bg-muted/15 text-muted`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `doc` | `any` | required | RxDB product document |
| `showQuantity` | `boolean` | `false` | Show quantity alongside status |
| `className` | `string` | — | Override badge container styles |

## Cart Components

### CartLine

Displays a single cart line item with product name, unit price, quantity, and line total.

```tsx
import { CartLine } from '@tallyui/components';

<CartLine item={{ doc: productDocument, quantity: 2 }} currencySymbol="$" />
```

Horizontal row with product info on the left (name + meta) and line total on the right.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `item` | `CartLineItem` | required | Object with `doc` and `quantity` |
| `currencySymbol` | `string` | `'$'` | Currency symbol |
| `className` | `string` | — | Override container styles |

### CartTotal

Displays a cart summary with subtotal, optional tax line, and total.

```tsx
import { CartTotal } from '@tallyui/components';

<CartTotal items={cartItems} taxRate={0.1} taxLabel="GST" />
```

Renders rows for subtotal and total, with an optional tax line when `taxRate > 0`. The total row has a top border separator.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `CartLineItem[]` | required | All cart line items |
| `currencySymbol` | `string` | `'$'` | Currency symbol |
| `taxRate` | `number` | `0` | Tax rate as decimal (0.1 = 10%) |
| `taxLabel` | `string` | `'Tax'` | Label for tax line |
| `className` | `string` | — | Override container styles |
